How to Use the Fetch API for Web Requests

The Fetch API provides a modern way to make HTTP requests in JavaScript. It is simpler and more powerful than older methods like XMLHttpRequest.

What is Fetch API?

The Fetch API allows us to make network requests asynchronously and handle responses using Promises.

Example:

    
fetch('https://api.example.com/data')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));

Why Use Fetch API?

  • Simpler syntax
  • Supports JSON parsing
  • Can handle different HTTP methods like GET, POST, PUT, DELETE

Making a POST Request

    
fetch('https://api.example.com/data', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ name: 'John Doe' })
})
  .then(response => response.json())
  .then(data => console.log('Success:', data))
  .catch(error => console.error('Error:', error));
